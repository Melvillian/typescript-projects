# @melvillian/common-lib

## Overview

Shared utility library providing formatters (bytes, counts, seconds), math helpers (distributions, rounding, percentages), and general utilities (error handling, object helpers, sleep, camelCase).

## Commands

- `bun run build` - Compile TypeScript
- `bun run dev` - Watch mode compilation
- `bun run typecheck` - Type check without emitting

## Key Files

- `src/formatters/` - Unit formatting (bytes, counts, seconds)
- `src/math/` - Math utilities (rounding, distributions, percentages)
- `src/utilities/` - General helpers (sleep, toCamelCase, errorToString)
